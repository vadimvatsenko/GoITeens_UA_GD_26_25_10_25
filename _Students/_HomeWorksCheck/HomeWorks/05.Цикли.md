Завдання: Гра "Вгадай число"



Мета:

Вивчити використання циклів і умовних операторів у C#.
Розвинути логічне мислення і навички програмування.


Опис:

Створіть просту гру на C#, де комп’ютер "загадує" число від 1 до 100, а гравець має вгадати його. Після кожної спроби комп’ютер має казати, чи є вгадане число більшим, меншим або рівним загаданому.

Вимоги:

Використовуйте цикл `while` або `do-while` для організації повторення спроб гравця.
Застосовуйте умовні оператори if, else if, else для перевірки вгаданого числа.
Гра має продовжуватися, доки гравець не вгадає число або не вирішить завершити гру.

Підказка:
Random random = new Random();

int secretNumber = random.Next(1, 101); // Комп'ютер загадує число

Кроки:

Ініціалізуйте змінну для зберігання загаданого числа.
Створіть цикл, де користувач буде вводити свої варіанти.
В циклі використайте умовні оператори для перевірки, чи вгадане число більше, менше, чи рівне загаданому.
Дайте користувачеві можливість завершити гру за бажанням.
Коли число вгадано, виведіть вітання і кількість спроб.
